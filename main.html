Page Replacement Algorithms

A modern, interactive visualization of FIFO, LRU and Optimal algorithms with step-by-step memory views, live statistics and comparisons – built by Luvkesh & Abhijeet.

Start Simulation Read the Theory
01 · Visual
Frame-by-Frame View

Watch how each page reference changes the frames. Hits and faults are colour-coded for instant understanding during explanation in class.

02 · Analytics
Live Comparison

Page faults, hits and fault rate are automatically calculated for FIFO, LRU and Optimal and summarized with a bar chart and a “best algorithm” highlight.

03 · Teaching
Step Sync Slider

A single slider highlights the same step across all three algorithms, making it very easy to demonstrate how each algorithm behaves differently for the same reference string.
